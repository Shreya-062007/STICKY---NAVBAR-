
🌐 Responsive Sticky Navbar with Hero Banner
This project demonstrates a modern responsive website layout featuring a sticky navigation bar and a full-screen hero banner. Built with clean and minimal CSS, it highlights smooth transitions, flexible alignment, and adaptive design principles.

✨ Key Features
Sticky Header: The navigation bar stays fixed at the top for easy access.

Dynamic Styling: Header background and text colors change on scroll for better readability.

Responsive Flexbox Layout: Ensures consistent alignment across devices.

Hero Banner: A full viewport background image (image.jpg) creates a striking first impression.

Smooth Transitions: Subtle animations enhance user experience.
